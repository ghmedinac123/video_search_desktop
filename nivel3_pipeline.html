② YOLO detecta y recorta objetos

YOLO analiza cada frame y detecta personas, vehículos, animales, etc. Por cada objeto detectado, recorta (crop) la región de la imagen donde está ese objeto. Un frame con 3 personas genera 3 recortes individuales.

④ ChromaDB almacena todo

ChromaDB

Se guarda el embedding vectorial de CLIP + la descripción textual de Gemini + toda la metadata. La imagen NO se guarda aquí, solo su ruta en disco.

# Lo que se almacena POR CADA CROP:

{
  "id": "cam01_20250225_142304_crop001",

  // Vector de CLIP (para búsqueda vectorial)
  "embedding": [0.234, -0.891, ...], // 768D

  // Metadata (para filtrar y recuperar)
  "metadata": {
    "camera": "cam01",
    "timestamp": "2025-02-25T14:23:04",
    "class": "person",
    "confidence": 0.94,
    "crop_path": "/crops/cam01/.../crop_001.jpg",
    "frame_path": "/frames/cam01/.../frame_042.jpg",
    "video_source": "/videos/cam01_2025-02-25.mp4",
    "video_second": 84,

    // Descripción de Gemini (para búsqueda textual)
    "description": "Mujer adulta, camisa roja manga corta, jean azul oscuro, bolso negro..."
  }
}

💡 ¿Por qué guardar AMBOS (embedding + descripción)?

Porque tienes dos formas de buscar:

1. Búsqueda vectorial (CLIP): "mujer camisa roja" → se convierte a vector → ChromaDB busca los vectores más cercanos. Bueno para búsquedas visuales generales.

2. Búsqueda textual (Gemini): Buscas en el campo "description" por palabras clave como "bolso negro" o "caminando hacia la derecha". Bueno para detalles específicos que CLIP podría perder.

Puedes combinar ambas búsquedas para resultados mucho más precisos.

El usuario ve una galería de imágenes recortadas de personas que coinciden con su búsqueda, cada una con: el crop de la persona, el frame completo, la descripción de Gemini, timestamp, cámara, y un botón para reproducir el video en ese momento exacto. Todo como si fuera WizSeeK de Dahua, pero open source.

Costos: ¿Gemini API vs LLM Local?

✅ Sí, usar Gemini 2.5 Flash por API es la mejor opción para tu caso

No necesitas correr un LLM localmente. Gemini Flash es barato, rápido, y genera descripciones mucho mejores que un modelo local como LLaVA. Todo lo demás (YOLO, CLIP, ChromaDB) corre 100% local en tu PC.
